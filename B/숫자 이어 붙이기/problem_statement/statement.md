# 숫자 이어 붙이기

##### 시간제한 1초, 메모리 제한 256MB

## 문제

자연수 N개가 주어졌을 때, 주어진 모든 자연수를 차례대로 이어 붙여 출력하는 프로그램을 작성하시오.



## 입력

첫 번째 줄에 자연수의 개수 N(1 ≤ N ≤ 10)이 주어진다.

두 번째 줄부터 N개의 줄에는 각 줄마다 자연수가 하나씩 주어진다. 모든 자연수는 1 이상 10,000 이하이다.



## 출력

주어진 모든 자연수를 차례대로 이어 붙여 출력한다.



## 예제 입력 1

```
3
12
345
6789
```



## 예제 출력 1

```
123456789
```
